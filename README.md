Проектування реляційної моделі даних (SQL) та бізнес-аналіз продажів (Power BI) на базі датасету Superstore.

# 🏢 Superstore: Проектування БД та Аналітичне Моделювання

## 📌 Огляд проекту
Цей проект демонструє повний цикл роботи з даними: від проектування реляційної моделі в SQL до побудови інтерактивного бізнес-дашборду в Power BI. Основна мета — перетворити сирі транзакційні дані мережі Superstore на структуровану базу даних для ефективного аналізу продажів та прибутковості.

## 🛠 Технічний стек
* **SQL (MySQL):** Проектування схеми "Зірка" (Star Schema), нормалізація даних, написання DDL та DML запитів.
* **Power BI:** ETL-процеси (Power Query), моделювання зв'язків та візуалізація ключових метрик (KPI).
* **Excel/CSV:** Робота з первинним датасетом.

## 🏗 Архітектура бази даних

Проект реалізовано на базі реляційної моделі (Star Schema). Окрім основних таблиць-вимірів, структура містить дві таблиці фактів для забезпечення стабільності аналітики:

### Схема бази даних (ER-діаграма):
![Database Schema](sql/db_schema.jpg)

### Опис таблиць:
* **`Orders` (Основна):** Центральна таблиця фактів, що містить фінальні очищені дані про продажі, прибуток та логістику. Вона підключена до Power BI для побудови звітів.
* **`Orders_test`:** Спеціальна таблиця для тестування нових запитів, перевірки цілісності даних та проведення A/B аналізу перед внесенням змін в основну модель.
* **Виміри:**
    * `Geography` — територіальна приналежність замовлень.
    * `Customers` — профілі та сегменти клієнтів.
    * `Products` — повний довідник товарів.

### Логіка обробки даних:
Дані спочатку завантажуються в технічну таблицю `temp_superstore`, проходять процес очищення та розподіляються між таблицями-вимірами. Використання окремої таблиці `Orders_test` дозволяє валідувати розрахунки (наприклад, перевірку сум після JOIN), не впливаючи на основний дашборд.

## 📊 Ключові показники (KPI) в аналізі
В межах Power BI звіту реалізовано відстеження наступних метрик:
* **Profitability Analysis:** Аналіз чистого прибутку за регіонами та категоріями товарів.
* **Shipping Efficiency:** Порівняння ефективності різних способів доставки (Ship Mode).
* **Geospatial Insights:** Візуалізація продажів на карті для виявлення найбільш прибуткових ринків.
